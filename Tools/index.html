Chú thích:
- Kết quả trả về: '1' thì là thành công hủy token.
- Có thể thử hủy thêm lần nữa để kiểm tra lại xem có được trả về kết quả: '190 Error validating access token: The session has been invalidated because the user changed their password or Facebook has changed the session for security reasons. (452) method auth.expireSession access_token --sanitized-- destruction_token --sanitized-- 452' nếu có tức là đã bị hủy từ trước đó.
- Còn nếu kết quả trả về là '101 Invalid application ID. (101) method auth.expireSession access_token destruction_token --sanitized-- 0' thì vui lòng kiểm tra lại xem đã nhập token hay chưa, hoặc có thể token chưa đủ ký tự.
